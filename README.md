### Run the project

```shell
# This assumes you have Ruby & RVM installed. If you don't, visit:
# https://rvm.io/ to install RVM.

# Install gem dependencies 
$ bundle install

# Start the server
$ rackup
```

Use your browser to visit `http://localhost:9292`
